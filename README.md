## Requirements
- Python 3.7.13
- Pytorch 1.10.0
- torchvision 0.11.0

We conduct the experiments with 4 24G RTX3090 on CUDA 11.1. For more details, please refer to the `requirements.txt`. 
```
pip install -r requirements.txt
```
